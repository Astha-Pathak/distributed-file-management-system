# Distributed File Management System

A scalable distributed file management system built using Python, gRPC, Docker, and microservice architecture.

## Features

* Distributed file storage
* Chunk-based file management
* Replication and fault tolerance
* gRPC-based communication
* Metadata management
* Multi-node storage architecture
* Dockerized deployment

## Tech Stack

* Python
* gRPC
* Protocol Buffers
* FastAPI
* Docker
* Linux

## Architecture

```text
Client → Gateway → Metadata Server → Storage Nodes
```

## Project Structure

```text

```
